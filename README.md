### Response Shape

| Name | Type | Description |
| --- | --- | --- |
| fulfillmentText | string | Text to be shown on the sreen |
| fulfillmentMessages | Object | Collection of rich messages to show the user |
| outputContext | Objects | Collection of output contexts |
